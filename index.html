    Logic Explanation

    My array stores objects that contain the name and value of each piece of loot entered. I use a traditional for loop in my render function to loop through the array, add up the total value, and create the list items that show up on the page. I use conditional logic (if statements) when the buttons are clicked to make sure the party size is at least 1, the loot name is not empty, and the loot value isn't negative before updating the array or splitting the gold.
